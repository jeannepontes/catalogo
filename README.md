Site catálogo oficial de Jeanne Pontes, Terapeuta Sistêmica especializada em Constelação Familiar, Terapia Individual, de Casal e Familiar.

✨ Sobre o projeto
Site de página única (one-page) desenvolvido em HTML, CSS e JavaScript puro, sem dependências externas além de Google Fonts. Apresenta os serviços, pacotes e formas de contato da terapeuta.
Paleta: tons âmbar e dourado sobre fundo creme quente
Tipografia: Cormorant Garamond (serifada) + Jost (sans-serif)
Animações: scroll reveal, gradientes animados, micro-interações em hover
